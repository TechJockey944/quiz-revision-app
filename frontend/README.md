# Quiz Revision App Frontend

A simple study website built in the `frontend` folder.

## How to use

1. Open `frontend/index.html` in a browser.
2. Paste a prompt or upload text files.
3. Click **Analyze and Build Pathway**.
4. Choose the number of questions, set a time limit if you want, and optionally enable flashcards.
5. Take the quiz and review the adaptive pathway that focuses on your weak areas.

## Features

- Upload your notes or paste a prompt
- Automatically create a learning pathway
- Generate a simple essay that explains the topic clearly
- Build a multiple-choice quiz
- Generate flashcards for the main concepts
- Show an adaptive follow-up plan based on quiz results

4. Deploy!

See [Vercel Documentation](https://vercel.com/docs) for more details.
